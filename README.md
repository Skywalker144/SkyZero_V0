# SkyZero: AlphaZero Implementation in PyTorch

SkyZero is a concise and efficient implementation of the AlphaZero algorithm, supporting environments such as Tic-Tac-Toe and Gomoku.

## 1. Core Principles of AlphaZero

The core idea of AlphaZero is the combination of **Monte Carlo Tree Search (MCTS)** and **Deep Neural Networks (DNN)**.

*   **Dual-head Network**: Uses a ResNet-structured neural network to output:
    *   **Policy ($p$)**: Predicted probabilities for each legal action in the current state.
    *   **Value ($v$)**: An estimate of the win rate for the current state (ranging from -1 to 1).
*   **Zero Knowledge**: Does not rely on human game records or heuristic evaluation functions. It starts from random weights and learns through self-play based solely on game rules.

## 2. Algorithm Flow

AlphaZero's training is an iterative process consisting of three main steps:

1.  **Self-Play**:
    *   The agent selects moves using MCTS.
    *   MCTS uses the current neural network to guide the search process (reducing search width and performing state evaluation).
    *   Records the game states, the probability distributions generated by MCTS, and the final outcome.
2.  **Data Collection**:
    *   Stores experience data from self-play into a **Replay Buffer**.
3.  **Optimization**:
    *   Samples data from the replay buffer to train the neural network.
    *   **Objective Function**: Makes the predicted policy $p$ close to the MCTS probabilities and the predicted value $v$ close to the actual game outcome.
    *   $L = (z - v)^2 - \pi^\top \log p + c \|\theta\|^2$

## 3. Project Structure

*   `alphazero.py` & `alphazero_parallel.py`: AlphaZero training logic (supporting parallelization).
*   `nets.py`: ResNet network definitions.
*   `envs/`: Game environment definitions.
*   `tictactoe/` & `gomoku/`: Training and play scripts for specific games.
*   `playgame.py`: General game-playing utility.

## 4. Quick Start

### Training the Model
Taking Tic-Tac-Toe as an example:
```bash
python tictactoe/tictactoe_train.py
```

### Playing Against the Model
```bash
python tictactoe/tictactoe_play.py
```
